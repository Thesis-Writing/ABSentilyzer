Single Input Classification

A guide on how to use the single input classification

The user may enter one or more sentences in the textfield, which has a maximum length of 300 characters. When the "Classify" button is pressed, the input is passed to the backend for processing. The user will only see the loading symbol during the classification process, indicating that the system is still processing the input.

Multiple Input Classification

A guide on how to use the multiple input classification

For the Multiple Input Classification, the user can upload a CSV file containing a maximum of 500 rows to be classified. After uploading the file and meeting the the requirements, the user can click the "Classify" button. The file is then passed to the backend for processing. The user will only see the loading symbol during the classification process, indicating that the system is still processing the file.

Renaming CSV Column Header

  • Users must rename the column which they want to classify in the CSV as "input".

    1. The column should contain sentiments about an entity, product, or services, which in the picture below is the column "Comments".

    2. The user must rename the column "Comments" into "input", then save the file.
